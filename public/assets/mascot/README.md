# Mascot Assets

The DJ-booth component is set up for swappable sprite frames. Final art should be exported as transparent PNGs with matching dimensions:

- `idle-1.png`
- `idle-2.png`
- `idle-3.png` optional
- `idle-4.png` optional
- `flourish-1.png` optional track-change frame

This Step 1 build uses `idle-1.svg` as a lightweight placeholder. Replace the component source with the PNG frame sequence during the interactive scene pass.
